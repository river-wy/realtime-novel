## 1. 项目路由

### `GET /api/projects` — 列出项目

```bash
curl "http://localhost:7778/api/projects?limit=20&offset=0"
```

**响应**：
```json
{
  "total": 3,
  "projects": [
    {
      "id": "abc123",
      "name": "我的小说",
      "palette": "urban_fantasy",
      "chapter_count": 5,
      "last_updated": "2026-06-17T10:30:00"
    }
  ]
}
```

### `POST /api/projects` — 创建项目

```bash
curl -X POST "http://localhost:7778/api/projects" \
  -H "Content-Type: application/json" \
  -d '{"name": "我的小说", "palette": "urban_fantasy", "initial_prompt": "一个关于..."}'
```

### `GET /api/projects/{id}` — 加载项目详情

```bash
curl "http://localhost:7778/api/projects/abc123"
```

### `DELETE /api/projects/{id}?confirm=true` — ⚠️ 删除项目（软删）

```bash
curl -X DELETE "http://localhost:7778/api/projects/abc123?confirm=true"
```

**注意**：`confirm` 必须为 `true`，否则返回 400。

## 2. 章节路由

```bash
curl "http://localhost:7778/api/projects/abc123/chapters"
```

### `GET /api/projects/{id}/chapters/{n}` — 读章节

```bash

```

```bash
curl -X POST "http://localhost:7778/api/projects/abc123/chapters" \
  -H "Content-Type: application/json" \

```

**注意**：60-100s 端到端，建议用 WebSocket 流式（见 §4）。

## 4. WebSocket 主对话（**流式**）

### `WS /api/chat` — 与「小说家」Agent 对话

```python
import asyncio
import websockets
import json

async def chat():
    uri = "ws://localhost:7777/api/chat"
    async with websockets.connect(uri) as ws:
        # 1. 发用户消息
        await ws.send(json.dumps({
            "type": "user_message",
            "content": "帮我生成下一章",
            "project_id": "abc123",
        }))
        # 2. 收 4 类事件
        while True:
            event = json.loads(await ws.recv())
            print(event)
            if event["type"] in ("agent_message", "error", "interrupted"):
                break

asyncio.run(chat())
```

**4 类事件**：
- `agent_thinking` — Agent 思考过程
- `tool_calling` — 工具调用（`{tool, args}`）
- `tool_result` — 工具结果（`{tool, result}`）
- `agent_message` — Agent 最终回复
- `error` — 错误
- `interrupted` — 被打断

**打断**：发 `{"type": "interrupt"}` 消息取消当前生成。

## 6. 错误码速查表

| 状态码 | 含义 | 场景 |
|---|---|---|
| 200 | OK | 成功 |
| 201 | Created | POST 创建项目成功 |
| 400 | Bad Request | 参数错误 / `confirm=false` |
| 404 | Not Found | 项目/章节不存在 |
| 409 | Conflict | 项目名重复 / 并发生成第 2 个 |
| 422 | Validation Error | Pydantic schema 校验失败 |
| 500 | Server Error | LLM 失败 / DB 失败 |

## 7. 鉴权

**v0.4 单机单用户，无鉴权**。所有路由开放访问。

## 8. 性能说明

| 路由 | 响应时间 |
|---|---|
| 9 个查询/操作类 | < 2s |
| `POST /chapters` | **60-100s**（生成章节） |
| `POST /image` | **< 30s**（Gemini 文生图） |
| `POST /onboarding`（5 步总） | < 25s |
| `WS /api/chat` 思考流式 | < 500ms/事件 |

## 9. CORS

v0.4 开放所有 origin（开发模式）。生产环境应限制为前端域名（如 `http://localhost:7777`）。

## 10. 示例：完整流程

```python
import asyncio
import websockets
import json
import httpx

async def full_flow():
    base = "http://localhost:7778"
    # 1. 创建项目
    r = httpx.post(f"{base}/api/projects", json={"name": "测试", "palette": "fantasy"})
    project_id = r.json()["id"]
    print(f"创建项目: {project_id}")
    # 2. 5 步 onboarding
    for step in ["1a", "1b", "2", "3", "4", "5"]:
        httpx.post(f"{base}/api/projects/{project_id}/onboarding",
                   json={"step": step, "payload": {}})
    # 3. WS 生成第 1 章
    async with websockets.connect(f"ws://localhost:7777/api/chat") as ws:
        await ws.send(json.dumps({
            "type": "user_message",
            "content": "生成第 1 章",
            "project_id": project_id,
        }))
        while True:
            ev = json.loads(await ws.recv())
            if ev["type"] == "agent_message":
                print(f"第 1 章: {ev['content'][:200]}...")
                break
    # 4. HTTP 读章节
    r = httpx.get(f"{base}/api/projects/{project_id}/chapters/1")
    print(f"字数: {r.json()['word_count']}")
    # 5. 软删
    httpx.delete(f"{base}/api/projects/{project_id}?confirm=true")

asyncio.run(full_flow())
```
